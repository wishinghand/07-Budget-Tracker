# Monthly Budget Calculator

## Technologies Used
- HTML
- CSS
    + SASS
    + Bulma.io
- Javascript
    + Angular
    + Gulp

## How to Run:
Clone the repo, `cd` into the directory and run:
```
npm install
```

and then serve it with the command:
```
gulp serve
```
